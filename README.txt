Saya Andika Tumanggor dengan NIM 1908107010052 merupakan mahasiswa Informatika pada Universitas Syiah Kuala. Disini saya mengerjakan tugas UAS Praktikum APL yaitu mengimplementasi pattern dengan benar. Pattern yang saya harus kerjakan ialah observer pattern dan decorator pattern. Dikarenakan nama saya Andika Tumanggor, berawal huruf A pada nama panggilan Andika. Jadi saya memilih topik Anime yang berawal huruf A.

Deskripsi : 
Anime sudah menjadi hal yang mendunia yang sudah jarang ditemui orang yang tidak mengetahui anime. Orang yang gemar dan tergila-gila dengan anime biasa disebut dengan wibu. Jika hal tersebut salah, maka penulis meminta maaf. Adapun pada penerapan patternnya, wibu akan mendapatkan notifikasi dari anime yang akan rilis, anime yang rilis tersebut akan diberikan juga jadwal lengkapnya.

Design pattern :
1. Observer pattern, pada observer pattern ini terdapat subject dan observer. Dimana yang menjadi subject ialah mimin dan observernya yaitu wibu.
2. Decorator pattern, pada decorator pattern anime yang awalnya dikatakan rilis pada kisaran awal bulan akan ditambah dengan jadwal yang lebih spesifik yaitu tanggal perilisan anime tersebut.

Compile & Run Program : 
1. Clone Repository
  $ git clone https://github.com/andikatmg07/LAB_APL_PatternAnime.git

2. Compile The Code
Go to the directory where the file is stored in cmd/terminal and run the following command:
     $ javac Main.java

3. Run the program!
    $ java Main
